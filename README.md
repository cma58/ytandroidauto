# YT Android Auto

Een Android-app waarmee je YouTube-muziek en -video's kunt afspelen via **Android Auto** — zonder officiële YouTube API, zonder trackers, volledig privé.

Gebouwd op de basis van [NewPipe](https://github.com/TeamNewPipe/NewPipe) met toegevoegde Android Auto-integratie.

---

## Functies

- 🚗 **Android Auto** — blader door geschiedenis en afspeellijsten op het scherm van je auto
- 🔍 **Zoeken** — zoek direct op YouTube via het Android Auto-zoekscherm
- 📋 **Afspeellijsten** — al je bladwijzers en lokale afspeellijsten zichtbaar in de auto
- 🕓 **Geschiedenis** — snel je recent bekeken nummers terugvinden
- 🔵 **Bluetooth auto-play** — verbindt je auto? Muziek hervat automatisch
- 🚫 **Geen API-sleutel** — gebruikt NewPipe Extractor, volledig privé
- ⬇️ **Downloads** — sla video's op voor offline afspelen
- 📺 **Achtergrond afspelen** — scherm uit, muziek aan

---

## Android Auto instellen

1. Installeer de APK op je telefoon
2. Verbind je telefoon via USB of draadloos met je auto
3. Open Android Auto → kies **YT Android Auto** als muziekapp
4. Blader via het aanraakscherm van de auto door geschiedenis, afspeellijsten en zoekresultaten
